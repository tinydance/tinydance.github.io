Project Overview

As machine learning practitioners continue finding new ways to create art, introducing these innovations to the art community becomes essential. Our project aims to do just that: make it easy for choreographers and dancers to use machine learning models that create dance from music.

Machine Learning - A Brief Introduction

To understand how we can get answers to our dance-related questions, we need to begin with machine learning.

What is machine learning and why do we care?

Understanding what machine learning is and why we choose to use it makes more sense if we compare it to “traditional programming”. Think of traditional programming as giving the computer exact instructions to follow. We set inputs and tells the computer what operations to perform, expecting to get certain results. For example, we give the computer the instructions to add and the inputs 1 and 2, expecting the computer to give us a 3 in return.
